JevaEngine - Update
Thursday, October 04, 2013
  • Imported new multiplayer map to enable online play. Servers will be on for three hours starting at 10:00AM GMT Pacific Time.
  • Servers may be under stress causing 'twitchy' player movement. More effective client-side prediction implementation is planned for next week to solve this.


JevaEngine - Update
Thursday, October 03, 2013
  • New preloader keeps JevaEngine up to date
  • Resolved bug where two interacting entities could carry out interaction even if one was removed from the world
  • Imported graphics assets from abandoned project PARPG
  • Resolved render order bug, render ordering is much more clever now
  • Discovered and resolved thread deadlock error that may have occured in clients\server
  • Added graphics configuration dialogue for JevaEngine.
  • Rhino JavaScript Script-Engine is now distributed with JevaEngine, resolving issues with some JRE distributions.
  • Resolved bug with audio subsystem not properly allocating clip instances from cache, causing them to effectivley be weakly referenced by the main Audio subsystem.
  • Resolved formatting issues in game's menu world.
  • Fixed bug which caused game resolution to incorrectly scale render buffer.
